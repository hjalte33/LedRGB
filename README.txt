  RGB.h - Library for controlling an rgb led. 
  It has at the moment 3 modes: static, blinking beteen 2 to 10 different colors, and fading between the same 2 to 10 different colors. 
  
  It only updates when the update function is runned.
  
  First created by Hjalte Nielsen December 2016.
  Github link: https://github.com/hjalte33/ledRemote.git
  Demo video on https://youtu.be/jiGEJYM5Z0I

  Feel free to make pull requests if you want to collaborate and make improvements.
